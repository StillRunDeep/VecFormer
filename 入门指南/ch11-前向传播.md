## 11.1 假设场景

为了追踪方便，假设：

```
batch_size B = 2        （两张图）
图1有 300 条线段（N1=300）
图2有 500 条线段（N2=500）
总线段数 N = N1 + N2 = 800

类别数 num_classes = 35
骨干最终特征维度 = 64
CAD 解码器嵌入维度 = 256
```

## 11.4 阶段二：PointTransformerV3 骨干网络

**代码位置**：`point_transformer_v3/model.py → PointTransformerV3.forward()`

### 步骤 1：特征嵌入（输入投影）

```python
# point.feat: (800, 7) → Linear(7, 32) → (800, 32)
point.feat = input_embedding(point.feat)   # (800, 32)  [enc_channels[0]=32]
```

### 步骤 2：编码器层 1（depth=2, channels=32）

```python
# 2个 Block，每个 Block：
# - CPE（稀疏卷积位置编码）：(800, 32) → (800, 32)
# - SerializedAttention：按 Hilbert 曲线排序，patch_size=1024（800<1024，全局注意力）
#     qkv = Linear(32, 96)    → (800, 96)
#     reshape → (800, 3, 4, 8)  [3=qkv, H=4头, D=8每头维度]
#     flash_attn → (800, 32)
# - MLP FFN：Linear(32,128)→GELU→Linear(128,32)

# 编码器层1 输出：
point.feat: (800, 32)    # 形状不变，但特征更丰富
```

### 步骤 3：编码器层 2（下采样，stride=2，channels=64）

```python
# GridPool 下采样：800条线 → 400条关键线（按体素选代表）
# 每个 2×2 体素区域选出 1 条代表线段

# 特征投影：Linear(32, 64)
# 2个 Block（类似层1，但 channels=64，H=4头，D=16每头）

# 编码器层2 输出：
point.feat: (400, 64)
point.offset: [0, 150, 400]   # N1→150, N2→250，各缩小约2倍
```

### 步骤 4：编码器层 3（stride=2，channels=128）

```python
# 400 → 200 条线
point.feat: (200, 128)
```

### 步骤 5：编码器层 4（stride=2，channels=256，depth=6）——最深层

```python
# 200 → 100 条线
# 6个 Block（最深，捕获最高级语义）
point.feat: (100, 256)
```

### 步骤 6：编码器层 5（stride=2，channels=512）

```python
# 100 → 50 条线
point.feat: (50, 512)
# 到达最高层，特征最抽象，分辨率最低
```

### 步骤 7-10：解码器（逐步上采样）

```python
# 解码器层 1：上采样 50→100，拼接编码器层4的 skip connection
# (50, 512) → Upsample(×2) → (100, 512+256=768) → Linear → (100, 256)
point.feat: (100, 256)   # [dec_channels[3]=256]

# 解码器层 2：100→200
# (100, 256) + skip(200, 128) → Linear → (200, 128)
point.feat: (200, 128)   # [dec_channels[2]=128]

# 解码器层 3：200→400
# (400, 64) + skip → Linear → (400, 64)
point.feat: (400, 64)    # [dec_channels[1]=64]

# 解码器层 4：400→800（恢复到原始分辨率）
# (400, 64) + skip(800, 32) → Linear → (800, 64)
point.feat: (800, 64)    # [dec_channels[0]=64]
```

**骨干输出**：

```python
backbone_feats: (800, 64)   # 每条线段对应一个 64 维特征向量
cu_seqlens:     (3,) = [0, 300, 800]   # batch 边界未变
```

## 11.5 阶段三：图层融合

**代码位置**：`modules/fusion_layer_feats_module.py`

```python
# 输入：backbone_feats (800, 64) + layer_ids (800,)
# 操作：按 layer_id 分组，组内特征平均

# 例：layer_ids = [2, 2, 2, 5, 5, 7, 7, 7, ...]
# 图层2的平均特征 = feats[layer_id==2].mean(0)   → (64,)
# 图层5的平均特征 = feats[layer_id==5].mean(0)   → (64,)
# ...

# 把组平均特征广播回每条线段
layer_avg_feats = scatter_mean(backbone_feats, layer_ids)  # (num_layers, 64)
fused = backbone_feats + layer_avg_feats[layer_ids]        # (800, 64)

# 输出：(800, 64)  形状不变，融入了图层上下文
```

## 11.6 阶段四：准备查询向量

**代码位置**：`modeling_vecformer.py → _init_queries()`

```python
# 推理时：所有线段都作为查询
queries = fused_feats                # (800, 64)
query_cu_seqlens = cu_seqlens        # (3,) = [0, 300, 800]

# 输入投影：Linear(64, 256)
queries = input_proj(queries)        # (800, 256)

# 骨干特征也投影到同一空间
backbone_proj = mask_proj(fused_feats)  # (800, 256)
```

## 11.7 阶段五：CAD 解码器（6个 Block）

**代码位置**：`cad_decoder/cad_decoder.py`

**一个 Block 的计算（以 Block 1 为例）**：

```python
# ── 输入 ─────────────────────────────────────────────────────
queries:    (800, 256)   查询向量
backbone:   (800, 256)   骨干特征（投影后）

# ── 自注意力 ─────────────────────────────────────────────────
# 查询向量内部互相关注
# Q = K = V = queries
q = Linear(256, 256)(queries)    # (800, 256)，拆成 8 头：(800, 8, 32)
k = Linear(256, 256)(queries)
v = Linear(256, 256)(queries)
attn_weights = softmax(q @ k.T / sqrt(32))  # (800, 800)，注意力权重
self_attn_out = attn_weights @ v             # (800, 256)

# Add & Norm
queries = LayerNorm(queries + self_attn_out)   # (800, 256)

# ── 交叉注意力 ────────────────────────────────────────────────
# 查询向量从骨干特征中提取信息
# Q = queries，K = V = backbone
q = Linear(256, 256)(queries)    # (800, 256)
k = Linear(256, 256)(backbone)   # (800, 256)
v = Linear(256, 256)(backbone)   # (800, 256)
attn_weights = softmax(q @ k.T / sqrt(32))  # (800, 800)
cross_attn_out = attn_weights @ v           # (800, 256)

queries = LayerNorm(queries + cross_attn_out)  # (800, 256)

# ── FFN ──────────────────────────────────────────────────────
ffn_out = Linear(256,1024)(queries)   # (800, 1024)
ffn_out = GELU(ffn_out)               # (800, 1024)
ffn_out = Linear(1024,256)(ffn_out)   # (800, 256)
queries = LayerNorm(queries + ffn_out)         # (800, 256)

# ── 预测头（iter_pred=True 时每层都预测）─────────────────────
# 语义预测
pred_sem = Linear(256, 35)(backbone_proj)      # (800, 35)

# 实例类别
pred_inst_labels = Linear(256, 35)(queries)    # (800, 35)

# 实例置信度
pred_inst_scores = Linear(256, 1)(queries)     # (800, 1)

# 实例掩码（内积）
pred_inst_masks = queries @ backbone_proj.T    # (800, 800)

# ── 输出 ─────────────────────────────────────────────────────
# queries 继续传入下一个 Block
queries: (800, 256)
# 本 Block 的预测附加到列表：
block_predictions.append({
    "pred_sem_labels":  pred_sem,          # (800, 35)
    "pred_inst_labels": pred_inst_labels,  # (800, 35)
    "pred_inst_scores": pred_inst_scores,  # (800, 1)
    "pred_inst_masks":  pred_inst_masks,   # (800, 800)
})
```

重复 6 次后，`block_predictions` 列表长度为 6，每个元素是一套完整预测。

## 11.8 阶段六：训练时的损失计算

**代码位置**：`criterion/criterion.py` + `criterion/instance_criterion.py`

```python
# 取所有 6 套预测（因为 iter_pred=True）
for block_idx, preds in enumerate(block_predictions):

    # ── 实例损失 ───────────────────────────────────────────────
    # 1. 匈牙利匹配（对 batch 中每张图分别做）
    #    图1（300条线，假设有 M1=5 个真实实例）：
    #    代价矩阵 (Q1=300, M1=5) → 匹配结果 [(pred_idx, gt_idx), ...]
    #
    #    图2（500条线，假设有 M2=8 个真实实例）：
    #    代价矩阵 (Q2=500, M2=8) → 匹配结果

    # 2. 类别损失（只对匹配到的查询）
    #    matched_preds = pred_inst_labels[matched_pred_idxs]  # (M1+M2, 35)
    #    class_loss = CE(matched_preds, gt_labels)

    # 3. 掩码损失（BCE + Dice）
    #    matched_masks = pred_inst_masks[matched_pred_idxs]   # (M1+M2, N)
    #    gt_masks: 真实掩码，形状也是 (M1+M2, N)
    #    bce_loss  = BCE(matched_masks, gt_masks)
    #    dice_loss = 1 - Dice(matched_masks, gt_masks)

    # 4. 置信度损失
    #    score_loss = MSE(pred_inst_scores[matched], iou_per_pred)

    # ── 语义损失 ───────────────────────────────────────────────
    # CE 对每条线段：
    # sem_loss = CE(pred_sem_labels, sem_ids)   # (800, 35) vs (800,)

    block_loss = (
        2.5 × class_loss +
        5.0 × bce_loss +
        5.0 × dice_loss +
        0.5 × score_loss +
        5.0 × sem_loss
    )

total_loss = mean(block_loss for all 6 blocks)
# 梯度通过 total_loss.backward() 反传
```

## 11.9 维度变化汇总表

| 阶段 | 操作 | 输入形状 | 输出形状 |
|------|------|---------|---------|
| 数据输入 | 拼接 batch | B 张图 | `(800, 7)` feats |
| PTv3 嵌入 | Linear(7→32) | `(800, 7)` | `(800, 32)` |
| Enc 层 1 | 2个Block | `(800, 32)` | `(800, 32)` |
| Enc 层 2 | 下采样+2Block | `(800, 32)` | `(400, 64)` |
| Enc 层 3 | 下采样+2Block | `(400, 64)` | `(200, 128)` |
| Enc 层 4 | 下采样+6Block | `(200, 128)` | `(100, 256)` |
| Enc 层 5 | 下采样+2Block | `(100, 256)` | `(50, 512)` |
| Dec 层 1 | 上采样+skip | `(50, 512)` | `(100, 256)` |
| Dec 层 2 | 上采样+skip | `(100, 256)` | `(200, 128)` |
| Dec 层 3 | 上采样+skip | `(200, 128)` | `(400, 64)` |
| Dec 层 4 | 上采样+skip | `(400, 64)` | `(800, 64)` |
| 图层融合 | GroupFeatFusion | `(800, 64)` | `(800, 64)` |
| 输入投影 | Linear(64→256) | `(800, 64)` | `(800, 256)` |
| CAD Dec×6 | Self+Cross+FFN | `(800, 256)` | `(800, 256)` |
| 语义预测 | Linear(256→35) | `(800, 256)` | `(800, 35)` |
| 实例掩码 | 矩阵乘法 | `(800,256)×(256,800)` | `(800, 800)` |
